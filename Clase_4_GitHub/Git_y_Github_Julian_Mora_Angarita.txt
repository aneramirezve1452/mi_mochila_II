Consigna | Git y Github
1. ¿Quién inventó el sistema de control de versión Git y por qué?

Git fue inventando por Linus Torvalds, porque antes de que existiera Git, se usaba Bitkeeper,
un sistema de control de versiones, que era distribuido y comercial; El hecho de que este fuera
un software propietario y algunos desacuerdos e inconformidades con su creador llevaron a torvarlds
a crear Git, un sistema de control de versiones de codigo abierto con la intención de gestionar 
proyectos grandes sin problemas

2. ¿A quién pertenece actualmente Github y por qué?

Actualmente Github pertenece a Microsoft, compañia que compro GitHub hace un par de años, esto es
